# AI-ML-PROJECTS
1. House Price Prediction

=>A machine learning project using Linear Regression to predict house prices based on features like area, bedrooms, and bathrooms. Demonstrates basic regression modeling and data visualization.

2. Student Score Prediction

=>A Multiple Linear Regression model that predicts student marks using study hours and sleep hours. Highlights how multiple factors influence performance.

3. Fake News Detection System

=>A text classification project using TF-IDF and Logistic Regression to detect whether news is real or fake. Demonstrates basic NLP techniques.

4. Face Recognition System

=>A real-time face recognition system built with OpenCV using Haar Cascade for detection and LBPH for recognition, with a simple GUI interface.
